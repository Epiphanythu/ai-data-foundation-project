# 中期汇报详细演讲稿

**项目**：基于多源数据的个人贷款违约风险检测
**演讲者**：同学 A + 同学 B
**总时长**：30 分钟

---

## 🎤 开场（30秒，同学 A）

```
大家下午好。

我们汇报的项目是：基于多源数据的个人贷款违约风险检测。

在开始之前，我想先问大家一个问题：
如果两个人收入相同、信用评分相同，
一个住在密西西比州，一个住在加州；
一个在2008年低利率时期贷款，一个在2006年高利率时期贷款；
他们的违约风险会一样吗？

这就是我们想回答的核心问题。

接下来，我将从研究动机开始，介绍这个项目。
```

---

## 第1部分：研究动机（3分钟，同学 A）

### 第1页PPT讲解

```
首先，为什么这个项目重要？

【指向PPT左侧 - 价值】
个人贷款违约直接影响金融机构的坏账损失，
也影响借款人的信用定价。
这是一个有实际价值的问题。

【指向PPT右侧 - 问题】
但传统的方法只看个人特征：
收入多少？信用评分多少？负债率多少？
这忽略了一个重要的现实：
同样的人，在不同的地区环境、不同的经济周期下，
违约风险可能完全不同。

【指向PPT中间 - 主张】
所以我们提出一个三层模型：
违约风险 = 个人特征 + 地区经济 + 宏观周期

但这带来了新的挑战：
第一，数据规模很大 - 我们要处理226万条贷款记录
第二，标签很复杂 - 很多贷款状态不能直接判断是否违约
第三，多源数据粒度不一致 - 个人、州级、季度级怎么对齐？

这就是我们项目要解决的核心问题。
```

---

## 第2部分：相关工作（3分钟，同学 A）

### 第2页PPT讲解

```
接下来，让我简要介绍一下相关工作。

【指向PPT左列 - 传统信用评分】
传统方法主要用FICO信用评分、收入、负债率这些个人指标。
优点是解释性强，大家都能理解为什么这个分数高或低。
但局限是：只看个人，忽略了环境和周期。

【指向PPT中列 - 机器学习】
机器学习用逻辑回归、随机森林、XGBoost这些算法。
优点是预测能力强，准确率高。
但局限是：容易变成黑盒，很难解释为什么这个人风险高。

【指向PPT右列 - 多源分析】
也有一些研究尝试加入地区收入、失业率、利率这些外部变量。
优点是视角更全面。
但局限是：数据对齐困难，口径不一致。

【指向PPT底部 - 我们的方法】
我们的定位是：
先把真实数据的获取、清洗、融合、统计分析做好，
建立可解释的洞察，
然后再考虑预测模型。
```

我们完成了三源数据融合：

47个季度的双重时间序列

现在，让我请同学B来介绍我们的核心发现。

## 第4部分：核心发现（同学 B 接手）

### 发现1：等级梯度（3分钟，同学 B）

```
谢谢同学A。

让我介绍我们的第一个核心发现：
贷款等级能够强区分违约风险。

【指向左侧图表】
大家请看这张图，这是Lending Club的贷款等级与违约率关系。

横轴是从A级到G级，纵轴是违约率。

我们可以看到一个明显的梯度：
A等级（信用最好）的违约率只有6.57%
G等级（信用最差）的违约率高达51.57%

【指向右侧关键数字】
这意味着什么？
G等级的违约率是A等级的8倍！

这个发现有两个重要意义：
第一，Lending Club的风险定价体系是有效的
第二，等级将成为我们后续分析的重要控制变量

当我们分析地区和宏观变量的影响时，
必须考虑不同州的等级结构是否不同。
```

### 发现2：利率相关（2分钟，同学 B）

```
第二个发现：利率与违约率高度相关。

【指向图表】
这张图展示了不同利率分箱的违约率。

我们可以看到：
低利率（小于8%）的贷款，违约率只有6.38%
高利率（大于等于24%）的贷款，违约率达到48.56%

这说明什么？
利率不是随机给出的。
平台在定价时已经考虑了风险因素。

这给我们的后续分析提出了一个要求：
当我们说"某个地区违约率高"时，
必须先问：是不是这个地区高风险贷款更多？

所以后续分析中，我们控制了利率和等级结构。
```

### 发现3：宏观环境相关（3分钟，同学 B）

```
第三个发现：宏观金融环境与违约率相关。

【指向图表】
这张图是我们将Lending Club的季度违约率
与联邦基金利率对齐后的结果。

蓝色线是违约率，红色线是联邦基金利率。
大家可以看到，两条线的趋势非常相似。

【指向相关系数】
我们计算了相关系数：0.844
这是一个很高的相关性。

但我要诚实说明：
这显示的是相关性，不能简单解释为因果关系。
可能是利率上升导致违约率上升，
也可能是经济不好导致两者同时上升。

不过，这个发现仍然很有价值：
它说明宏观环境值得纳入风险分析框架。

我们还做了年度分析和季度分析，
季度分析有47个时间点，比年度的12个点更可靠。
```

### 发现4：地区经济独立解释力（3分钟，同学 B）

```
第四个发现：地区经济有独立解释力。

【指向左侧图表】
大家首先看这张图：
横轴是州级贫困率，纵轴是州级违约率。
我们可以看到正相关趋势，相关系数是0.398。

但是，有人可能会质疑：
"某些州违约率高，是不是因为那里高风险贷款更多？"

【指向右侧图表】
为了回答这个问题，我们做了控制变量分析。
我们控制了州级的平均利率，重新计算了违约率残差与贫困率的相关性。

结果是：0.362，仍然显著。

我们还控制了州级的平均FICO信用分，
相关性反而上升到了0.459。

这说明什么？
地区经济变量有独立于个人特征的解释力。
即使两个人信用评分相同、利率相同，
生活在贫困率更高的州，违约风险仍然更高。
```

### 发现5：组合风险分层（2分钟，同学 B）

```
最后一个发现：组合风险分层能识别高风险人群。

【指向图表】
如果我们把等级和贷款用途组合起来看，
会发现更精准的风险分层。

【指向最高风险组合】
G等级 + debt_consolidation（债务整合）
违约率高达52.99%

这意味着什么？
每两笔这样的贷款，就有一笔会违约。

这个发现有实际应用价值：
风控部门可以直接识别这些高风险组合，
进行更严格的审核或定价。

这也说明：
单一维度的分析是不够的，
组合维度更能刻画真实风险。
```

## 💡 演讲技巧提示

### 1. 使用图表时的手势
```
• "大家请看这张图" - 用手指向图表
• "横轴是...纵轴是..." - 明确指出坐标轴
• "这里我们可以看到..." - 具体指出数据点
• "这个数字意味着..." - 解释数据意义
```

### 2. 过渡语
```
• "介绍了...之后，让我们看看..."
• "基于这个发现，我们进一步做了..."
• "这个发现引出了下一个问题..."
• "现在让我请同学B来介绍..."
```

### 3. 强调重点
```
• "请大家注意这个数字..." - 稍微放慢
• "这意味着什么？" - 停顿1秒
• "这是一个很重要的发现..." - 语气加重
• "我们要诚实说明..." - 建立信任
```

### 4. 时间控制
```
• 如果某页讲太快：多举一个具体例子
• 如果某页讲太慢：跳过详细解释，直接说结论
• 如果整体超时：跳过"奏效/受阻"这一页
• 如果时间充裕：在每个发现后加一个实际案例
```

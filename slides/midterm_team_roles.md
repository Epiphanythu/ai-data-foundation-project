### 演讲稿要点

**开场（30秒）**：
```
"大家下午好。我们汇报的项目是：
基于多源数据的个人贷款违约风险检测。

我想先问大家：两个收入相同、信用评分相同的人，
在不同州、不同利率环境下，违约率会一样吗？

这就是我们想回答的问题。"
```

**第1页 - 研究动机（3分钟）**：
```
[价值 - 1分钟]
"个人贷款违约直接影响金融机构坏账。
但传统只看个人特征，忽略了环境和周期。"

[问题 - 1分钟]
"违约风险 = 个人特征 + 地区环境 + 宏观周期
这个三层模型很难验证，因为：
• 数据规模大：226万条记录
• 标签复杂：很多状态不能直接判断
• 多源粒度不一致：个人、州、季度"

[过渡 - 1分钟]
"所以我们的目标是：
用真实数据，验证这个三层模型是否成立"
```

**第2页 - 相关工作（3分钟）**：
```
"现有方法有三类：

第一类：传统信用评分
• 用 FICO、收入、负债率
• 优点：解释性强
• 局限：偏重个体特征

第二类：机器学习预测
• 用 LR、RF、XGBoost
• 优点：预测能力强
• 局限：容易黑盒化

第三类：多源风险分析
• 加入地区、宏观变量
• 优点：视角全面
• 局限：数据对齐困难

我们的定位：
先做好数据清洗、融合、可解释分析
再考虑预测模型"
```

**第3页 - 数据全景（2分钟）**：
```
"我们完成了三源数据融合：

Lending Club：226万条记录，151个字段 ✅
FRED 宏观：47个季度时间点 ✅
USDA ERS：50个州经济数据 ✅
Home Credit：需要网页授权 ⚠️

融合成果：
• 136万已完结样本
• 50州完整匹配
• 47季度时间序列"
```

**第4页 - 发现1（3分钟）**：
```
"第一个发现：等级梯度明显

[指向图表]
大家可以看到，从A到G，违约率呈明显上升趋势：
• A等级：6.57%
• G等级：51.57%
• 相差了8倍

这说明平台的风险定价是有效的。
等级将成为我们后续分析的重要控制变量。"
```

**第5页 - 发现2（2分钟）**：
```
"第二个发现：利率与违约率高度相关

[指向图表]
低利率（<8%）违约率6.38%
高利率（≥24%）违约率48.56%

这说明利率已经包含了平台的风险判断。
后续分析地区和宏观变量时，必须控制利率。"
```

### 可能被问到的问题
- "为什么这个项目重要？" → 强调实际价值和三层模型创新
- "现有方法的局限是什么？" → 具体说明每种方法的不足
- "数据是如何收集的？" → 说明 Kaggle + 公开数据源
